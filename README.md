# ReferdBy

ReferdBy is a cross-platform referral and rewards application.

## Getting Started

To get started with local development, please refer to the [Cursor Migration & Local Development Guide](cursor-migration.md).

### Prerequisites

- Node.js (v18 or later)
- npm

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The application will be available at `http://localhost:8080`.

## Tech Stack

- **Frontend**: React, Vite, TypeScript
- **UI**: Shadcn UI, Tailwind CSS
- **Backend**: Supabase
- **Mobile**: Capacitor
